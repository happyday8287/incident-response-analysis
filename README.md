# Incident Response Analysis Project

## Overview

This project simulates a security incident involving repeated failed login attempts from a suspicious IP address.

## Detection

Suspicious activity was detected using SQL log analysis.

## Analysis

The IP address 10.0.0.50 showed repeated failed login attempts, indicating a potential brute-force attack.

## Response

* Block the suspicious IP address
* Monitor for further unusual activity
* Implement account lockout policies

## Conclusion

This project demonstrates how basic tools like SQL and Wireshark can be used to detect and respond to security incidents.
